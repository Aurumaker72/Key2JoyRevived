# `Keyboard.Simulate` (`Keys`, `PressState`)


Simulate pressing or releasing (or both) keyboard keys.


## Parameters

* **key (`Keys`)** 

	Key to simulate

* **pressState (`PressState`)** 

	Action to simulate
